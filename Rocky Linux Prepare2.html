Rocky Linux 安裝手冊 預備篇

-- 以 Rocky Linux 9.7 為例 --

Rocky Linux 有多種安裝方法, 比較簡單快速的是事前先備好 Rocky Linux 的安裝光碟. 由於科技的演進, 現在也可以將 DVD 碟片轉換為可開機 USB 隨身碟, 再以它來安裝.

  1. 下載 Rocky Linux 9.7 DVD 映像檔.

  2. 安裝 Rocky Linux 9.7

    • 使用 DVD 光碟: 直接燒錄 DVD 光碟.

    • 使用可開機 USB 隨身碟: 以免費軟體 Rufus, 將 DVD 映像檔製作成可開機 USB 隨身碟.

下載 Rocky Linux 9.7 DVD 映像檔說明如下:

  1. 下載 Rocky Linux 9.7 DVD 映像檔.

    • 開啟 browser 直接連至 Rocky Linux 官網的下載頁面 https://rockylinux.org/zh-TW/download.

    • 或者在 browser 在搜尋欄中輸入 Rocky Linux.

      Google Search 顯示的搜尋結果.

      請點擊 Rocky Linux 網站下方的 "下載".

  2. 選擇下載 "Rocky Linux 9"

    Rocky Linux 的中文下載頁面.

    預設影像的上方欄中選擇 AMD/Intel (x86_64) 按鈕, 並於選擇版本:下方欄中選擇 Rocky Linyx 9 按鈕. 結果如下圖:

    Rocky Linux 9.7 下載畫面.

    點選 DVD ISO 按鈕開始下載. (由於我們後續的安裝是選用最小安裝, 所以下載 Minimal ISO 也可以)

    附註一: 目前 Rocky Linux 9 最新的版本為 9.7 版, 下載其他的舊版本可以直接連接至 "https://download.rockylinux.org/pub/rocky/9/isos/x86_64/" 選擇想要下載的映像檔.

    附註二: 日後 Rocky Linux 9.7 再升版後, 想要再下載 9.7 版也可以直接連接至 "https://dl.rockylinux.org/vault/rocky/9.7/isos/x86_64/" 下載.
    (目前此連結是無效的)

  3. 下載後的檔案即可用於 DVD 光碟片之燒錄 (Minimal ISO) 或用於製作可開機 USB (DVD ISO, Minimal ISO).

下載 Rufus 說明如下:

Rufus 是免費軟體 (參見 Github 上的 Rufus FAQ:Why don't you sell Rufys?). 執行檔目前只有 Windows 版本, 也沒有語系之分 (即單一版本).

  1. 在 browser 在搜尋欄中輸入 "rufus 下載".

    Google Search 顯示的搜尋結果.

    請點擊第一個搜尋結果: Rufux 網站, 進入 Rufus 中文首頁.

  2. 進入 Rufus 中文首頁

    Rufus 中文首頁.

    請向下捲動到 "下載" 段落, 或者點擊選單列上的 "下載".

  3. 下載 Rufus

    Rufus 中文下載段落.

    選擇下載 "標準版" 或 "可攜版" 均可.

使用 Rufus 製作可開機 USB 說明如下:

請注意: 舊 PC 的 BIOS 可能不支援高速 USB Flash. 如果在舊 PC 上無法使用該 USB Flash 安裝作業系統時, 請改用舊款 USB Flash (一般容量在 16GB 以下) 重新製作.

Rufus 預設會使用目前 Windows 的語系, 要更動使用的語系可以進入後更改語系設定.

  1. 首次執行 Rufus, 會詢問 "是否啟用自動線上檢查更新". 可依需求開啟或關閉.

    詢問 "是否啟用檢查更新" 的對話框.

  2. 選擇要使用的 USB Flash.

    已插入 USB Flash 時的畫面.

    未插入 USB Flash 時的畫面.

    PC 上插入多個 USB Flash 可以選擇使用哪一個 USB Flash.

    附註三: Rocky Linux 9.7 版的 Minimal ISO 大小約 2.4GB, 故可以使用 4GB 之 USB Flash 即可. 但 DVD ISO 大小約為 13GB, 故請使用容量至少為 16GB 之 USB Flash (≥16GB).

  3. 載入 Rocky Linux DVD 映像檔.

    按下 選擇 按鈕 (前圖中標示 2 之按鈕), 並選擇先前下載之 Rocky Linux 映像檔.

    載入 Rocky Linux DVD 映像檔後之畫面.

  4. 選擇寫入模式.

    載入之後即可按下 執行 按鈕, 進行正式執行前之檢查. 即預備開始執行 Rocky Linux 開機 USB Flash 之製作.

    寫入模式之畫面.

    選擇預設 (建議) 之 ISO 映像模式寫入.

  5. 補載入 Grub 2.06.

    如畫面之說明: 此 PC 之 Rufus 未備有 Grub 2.06 需補下載.

    補載入 Grub 2.06 之畫面.

    此處需按 是 進行下載.

  6. 補載入 Syslinux 6.04.

    如畫面之說明: 此 PC 之 Rufus 未備有 Syslinux 6.04 需補下載.

    補載入 Syslinux 6.04 之畫面.

    此處需按 是 進行下載.

  7. 最後告警.

    開始執行寫入之告警畫面.

    按 OK 按鈕開始執行.
